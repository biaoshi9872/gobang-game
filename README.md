# 网页五子棋小游戏 (Gobang)

这是一个使用原生前端技术 (HTML + CSS + JavaScript) 编写的网页五子棋小游戏。

## 🎮 游戏功能
- 基础的五子棋落子规则（黑白交替落子，黑子先手）
- 自动判断胜负（横、竖、左斜、右斜方向任意五子连珠即获胜）
- 游戏状态提示（当前回合下子方显示）
- 一键重新开始功能
- 简洁美观的 UI，包含棋盘网格、星位以及棋子的立体渐变效果

## 🚀 使用说明
该项目为纯静态的网页项目，**无需安装任何 Node.js 依赖或构建工具**。

**方法一：直接运行 (最简单)**
在电脑文件夹中找到本项目目录，直接双击打开 `index.html` 文件，浏览器会自动加载并运行该游戏。

**方法二：本地服务器运行 (推荐开发者使用)**
如果你使用了 VS Code 等编辑器，并且安装了 `Live Server` 插件，可以右键点击 `index.html` 选择 "Open with Live Server" 运行，这样体验更贴近真实网站。

## 📁 目录结构
- `index.html`：页面的基础结构
- `style.css`：页面样式、棋盘颜色、光影效果
- `script.js`：核心游戏逻辑，包括 Canvas 绘图、点击事件处理与胜负判断算法
